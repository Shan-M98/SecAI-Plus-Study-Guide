# CompTIA SecAI+ CY0-001 Study Guide

Interactive practice exam and study materials for the CompTIA SecurityAI+ (CY0-001) certification.

## Study Resources

- **[Quizlet Flashcards](https://quizlet.com/1163244419/comptia-ai-flash-cards/)** - 110 flashcards for quick review

## Practice Exam

**[Open Practice Exam](practice_exam.html)** - Download and open in any browser. No server needed.

### Features
- **110 multiple-choice questions** covering all 4 exam domains
- **4 question types** randomized each attempt:
  - **SCENARIO** - Real-world situations, pick the correct concept
  - **IDENTIFY** - Given a definition, pick the correct term
  - **DEFINE** - Given a term, pick the correct definition
  - **ELIMINATE** - "Which does NOT belong to this domain?"
- **2h 45m countdown timer** matching the real exam
- **Auto-grading** with pass/fail (70% threshold)
- **Per-domain score breakdown**
- **Filter results** by Correct / Wrong / Skipped
- **Session persistence** - answers survive page refresh
- **Reshuffled on retake** - different order and question types each time

## How to Use

1. Download or clone this repo
2. Open `practice_exam.html` in your browser
3. Answer all 110 questions
4. Hit **Submit** to see your results
5. Review wrong answers and retake
